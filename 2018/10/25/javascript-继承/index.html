javascript 继承的实现

2018-10-25

继承是指一个对象直接使用另一对象的属性和方法。

Javascript 在ES6之前并没有像其他面向对象编程语言（如：C++、Java）有“类”(class)的概念，虽然设计了构造函数，但只能创建对象的副本，无法共享属性和方法。所以在ES6之前，js对象的继承主要还是通过prototype属性实现。
