JS 数组

发表于 2016-11-24 | 更新于: 2018-10-12
Javascript Array对象操作总结

注：

  1. 标记星号是会改变原数组的操作
  2. 元素操作索引从 0 开始
  3. 包含起始位置和结束位置的操作，结束位置都不包含在内，是前闭后开区间

concat（连接数组）

join（数组转为字符串）

*pop（删除最后一个元素）

*shift（删除第一个元素）

*push（末尾添加元素）

*unshift（首部添加元素）

*reverse（数组元素颠倒）

slice(start, end) （浅拷贝元素）

*splice （替换、删除元素）

*sort（排序）

a - b < 0，a 排在 b 前面

a - b = 0，a, b 相对位置不变

a - b > 0，b 排在 a 前面

*forEach（遍历）

map（映射）

filter（过滤）

some（是否包含）

返回true或false

every（全部通过）

返回true或false

find（返回第一个匹配元素）

findIndex（返回第一个匹配元素的索引）

indexOf(‘str’, start)（从开头查找元素）

返回第一个匹配元素索引或 -1

lastIndexOf(‘str’, start) （从末尾查找元素）

返回最后一个匹配元素索引或 -1

includes（是否包含元素）

参数：searchElement 要查找的元素，fromIndex 从该索引出开始查找

reduce（累加器）

回调函数包含四个参数：acc 累加器，cur 当前值，idx 当前元素索引，arr 原数组

最好给累加器一个初始值。

如果不给初始值，acc 默认值为数组第一个元素，cur 从数组第二个元素开始，idx 从 1 开始。

reduceRight (累加器，从右到左)

参数同 reduce 方法

isArray（是否是数组）

from （类数组转化为数组）

参数：arrayLike 类数组，mapFn 遍历函数

of (创建数组)

参数：数组的值

*copyWithin(target, start, end)（数组内浅复制元素）

参数：target 替换目标索引，start 开始复制索引，end 结束复制索引（不包含该位置）

复制后原数组被修改，数组长度维持不变

*fill （用一个固定的值填充数组）

参数：value 填充的值，start 起始索引， end 终止索引（不包含该位置）

entries（返回可迭代对象）

keys（返回包含每个索引的可迭代对象）

values（返回包含每个值的可迭代对象）

[@@iterator]() （iterator 方法）

返回值与 values 方法一样
